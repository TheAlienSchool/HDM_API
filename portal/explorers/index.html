Essential Concept

What is a Phase State?

A phase state is a stable configuration a system inhabits. Ice, water, steam—different phase states of the same substance. A dodecahedron is a phase state. A chaotic attractor is a phase state. Your consciousness right now is a phase state.

Each explorer shows you a different phase state, and the invisible parameters (dark matter) that shape it. When parameters shift slightly, the phase state holds steady. When they cross a threshold (bifurcation point), the system reorganizes totally.

You are moving through phase states as you spiral through this laboratory. Each steep shifts your parameters slightly. By session 7, you have crossed bifurcations invisible as you crossed them. You are not the same. The Bloom is the recognition of this.
