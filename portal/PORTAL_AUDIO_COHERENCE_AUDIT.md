## TECHNICAL NOTES

### **Deployment Pattern (Consistent Across All Pages)**
```html
<!-- In <head> -->
<script src="https://cdnjs.cloudflare.com/ajax/libs/tone/14.8.49/Tone.js"></script>
<script src="./js/[ENGINE_NAME].js"></script>
<script>
  let audioEngine = null;
  async function initEngine() {
    if (audioEngine) return;
    await Tone.start();
    audioEngine = new [EngineClass]();
    // Attach event listeners
  }
  ['click', 'keydown', 'mousedown', 'touchstart'].forEach(event => {
    document.addEventListener(event, initEngine, { once: true });
  });
</script>
```

- Web Audio API: All modern browsers
- Fallback: Silent graceful degradation (no errors)
