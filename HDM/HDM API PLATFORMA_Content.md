## SLIDE 4: THE PRODUCT — "The HDM API Platform"

**Headline:** HDM is the development intelligence infrastructure that every wellness and coaching app is missing.

**Body:** The HDM API Platform is a developer-facing infrastructure product. It accepts three types of input — text, behavioral signals, and self-assessments — and returns a **Development State Profile (DSP)**: a structured, real-time representation of where a user is across seven dimensions of human development.

**The Seven Dimensions:**
- Stillness Index — the capacity for present-moment awareness
- Curiosity Velocity — the rate and depth of exploratory engagement
- Relational Resonance — the coherence of interpersonal connection
- Creative Frequency — the originality and diversity of self-expression
- Trust Gradient — the velocity of trust-building with self and others
- Purpose Coherence — the alignment between stated goals and lived behavior
- Emergence Signal — the integration of all dimensions into a coherent development arc

**The Surface Tension Score:** A composite signal derived from the gap between Purpose Coherence and Trust Gradient. The higher the score, the more a user knows where they want to go but is struggling to get there — the precise condition that the most valuable interventions are designed to address.

**The output:** A real-time API response that tells any integrated application: *This user is here. They need this. Now.*
